Cadastro de Empresas

Esta é uma Single-Page Application (SPA) desenvolvida com Angular que acessa um back-end moderno.

O back-end segue padrões como:

  • DDD (Domain-Driven Design)
  • SOLID Principles
  • CQRS (Command Query Responsibility Segregation)
  • API RESTful com segurança implementada

Foco na escalabilidade, modularidade e boas práticas de desenvolvimento.
